# Merckomatic's Fresh Squanch

Android-only public release repo for Merckomatic's Fresh Squanch, a standalone
personal Spotify playlist companion.

The app is not affiliated with Spotify. Each user supplies their own Spotify
Developer Client ID, signs in with Authorization Code + PKCE, and keeps tokens
and music data on their Android device.

## Android

- Install and release notes: [android/README.md](android/README.md)
- Privacy notes: [android/PRIVACY.md](android/PRIVACY.md)
- Release checklist: [android/RELEASE_CHECKLIST.md](android/RELEASE_CHECKLIST.md)

Signed APKs are published through GitHub Releases. Updates are manual: install
the newer APK over the old one.
